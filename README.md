# Voron Trident

## General Features
- Voron Trident 300
- Will run [Danger Klipper](https://dangerklipper.io/)
- 300x300x??? build volume
- 300x300mm 750W 110V silicone bed heater
- BigTreeTech Octopus v1.1 MCU
- RaspberryPi 4
- BigTreeTech EBB36v1.2 CAN board
